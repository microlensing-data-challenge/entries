# Entries to the 2018 Microlensing Data Challenge

This repository documents the submitted entries for the 1st microlensing data challenge.  

A detailed description of the required contents for each entry can be found here:
http://microlensing-source.org/data-challenge-entry-contents/

The original challenge dataset is available from https://zenodo.org/records/18041245
and the team entry datasets can be found at https://zenodo.org/records/21235143
